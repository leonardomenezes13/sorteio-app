# Sorteio App

Aplicativo Android simples desenvolvido para treinar meus conhecimentos em desenvolvimento Android.

O projeto foi criado com o objetivo de praticar a construção de interfaces utilizando XML, programação com Kotlin e execução de aplicativos no emulador do Android Studio.

## Funcionalidades

- Exibição de uma tela simples de sorteio
- Botão para realizar o sorteio
- Exibição do resultado sorteado

## Tecnologias utilizadas

- Kotlin
- XML
- Android Studio
- ConstraintLayout

## Objetivo do projeto

Este projeto foi desenvolvido para praticar conceitos básicos de desenvolvimento Android, como:

- Criação de layouts com XML
- Manipulação de componentes da interface
- Uso de IDs para acessar elementos da tela
- Implementação de eventos de clique
- Execução e testes em um dispositivo virtual
- Versionamento de código com Git
